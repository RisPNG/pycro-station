Select Excel workbooks, read the `GRN Sum` sheet from each one, combine every data row under a single header row, then sort the result with non-`VN` warehouse rows first by `Doc Date` ascending and `VN` warehouse rows after them by `Doc Date` ascending. Visual formatting, column widths, hidden columns, row height, freeze panes, filters, and number formats are copied from the first valid selected `GRN Sum` sheet.
