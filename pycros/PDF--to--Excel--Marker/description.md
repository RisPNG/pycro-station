Uses marker-pdf to extract content from PDF files and convert them to Excel format. Supports both CPU and GPU processing with automatic detection of discrete vs integrated GPUs for optimal performance.

GPU Recommendations:
- Discrete GPU (4GB+ VRAM): Use GPU mode for faster processing
- Integrated GPU (<4GB VRAM): Use CPU mode to avoid out-of-memory errors
- No GPU: CPU mode works on all systems

The tool extracts tables and text content from PDFs, preserving structure where possible.
